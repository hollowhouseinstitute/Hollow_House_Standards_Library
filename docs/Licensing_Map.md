# Licensing Tier Alignment

## Public Repository Access
Includes:
- high-level standards descriptions
- conceptual frameworks
- governance posture
- non-operational documentation

Does NOT include:
- datasets
- applied workflows
- detailed scripts
- service deliverables

---

## Research Subscription (Web App)
Includes:
- expanded framework documentation
- methodological notes
- teaching materials
- licensed reference use

Restrictions:
- non-commercial only
- no operational deployment

---

## Institutional / Commercial License
Includes:
- applied use rights
- internal deployment permission
- audit and governance integration
- vector mapping and framework application

Granted:
- by request
- by contract
- manually approved
